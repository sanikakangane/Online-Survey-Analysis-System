# 😃 Online Survey Analysis System (C++)

## 📝 Overview
The Online Survey Analysis System is a simple console-based C++ program that collects user survey ratings and performs basic analysis. The system allows users to submit responses and view statistical results such as the average rating and total number of responses. 📊

This project demonstrates the use of fundamental C++ programming concepts including arrays, loops, conditional statements, and user input/output operations. 💻

## ✨ Features
- Submit survey responses 📝
- Accept ratings between 1 and 5 ⭐
- Store up to 100 responses 🗂️
- Calculate average rating 📈
- Display total number of responses 📋
- Menu-driven user interface 🎛️

## 🔄 Program Workflow
1. The program displays a menu with three options:
   - Submit Response 📝
   - View Analysis 📊
   - Exit 🚪

2. When the user selects **Submit Response**:
   - The user enters a rating between 1 and 5 ⭐
   - The rating is stored in an array 🗃️

3. When the user selects **View Analysis**:
   - The program calculates the average rating 📈
   - Displays the total number of responses 🧮

4. The program continues until the user chooses **Exit** 🚪

## 🛠️ Technologies Used
**Programming Language:** C++ 🖥️

**Concepts Used:**
- Arrays 
- Loops (do-while) 
- Conditional Statements (if-else) 
- Standard Input/Output (iostream)
